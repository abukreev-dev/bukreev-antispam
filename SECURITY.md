# Security Policy

## Supported Versions
Security fixes are provided for the latest release branch only.

| Version | Supported |
| ------- | --------- |
| 2.1.x   | Yes       |
| < 2.1   | No        |

## Reporting a Vulnerability
If you found a security issue, do not open a public GitHub issue.

Please report privately by email:
- security contact: `abukreev.dev@gmail.com`

Include:
- affected plugin version
- WordPress and PHP version
- reproduction steps
- impact assessment

Response targets:
- initial response: within 3 business days
- status update: within 7 business days
